O Startx é um tema Open Source (código aberto) para que você edite com suas informações e disponibilize seus projetos na web para download de maneira prática.

Galeria

Sobre

Acreditamos no poder do Open Source, por isso criamos uma tema já com layout estruturado para que você, desenvolvedor de softwares, web ou de qualquer outro tipo, possa editá-lo com as suas informações que considere importantes e disponibiliza-lo na web de maneira ágil, sem precisar ficar digitando muitos códigos ou perdendo tempo com questões de design. Já está tudo pronto!

Com o StartX você pode adicionar imagens do seu projeto, inserir as suas redes sociais ou do projeto, escrever um texto explicativo, direcionar para o download, elaborar um curto manual de como usar e ainda conta com um campo de contato para que os usuários possam escrever para você.

Abaixo faça o download do tema e confira!

Como usar?

Faça o download do tema

Faça o download do tema nesta página, logo acima. Clique no botão BAIXAR na área de DOWNLOAD. Descompacte o arquivo fonte e abra no seu editor de textos de preferência.

Experimente nossa ferramenta online, troque os textos e vizualise como pode ficar seu site antes mesmo de fazer o download. Veja aqui

Edite o tema

A maior vantagem de um projeto Open Source é poder edita-lo completamente. Trocar backgrounds, fontes, cores e até adicionar mais áreas e conteúdo. Edite da forma que preferir com suas informações.

Configurando seu projeto para download

Veja como é simples:
* Zipe seu projeto e renomeie para projeto.rar
* Coloque o na pasta arquivo
* Pronto! Ao atualizar a página e clicar no botao baixar o navegador fará o download de seu projeto.

Algumas Vantagens

* Projeto open souce totalmente editável.

* Código fonte comentado em Ingles e Português-BR.

* O StartX utiliza o framework bootstrap.

* Facilidade em disponibilizar o seu projeto para download na Web.

* Layout simples e rápido, pronto para divulgar o seu projeto!
